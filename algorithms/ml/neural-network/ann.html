Artificial Neural Network — Forward Pass & Backpropagation

A fixed 1 → 2 → 1 network trained on one example, every calculation shown step by step.

A neural network learns in three phases: (1) Forward Pass — compute predictions layer by layer, (2) Backpropagation — use the chain rule to find how much each weight contributed to the error, (3) Weight Update — nudge each weight in the direction that reduces the error.
SymbolMeaning
zWeighted sum  z = w·a + b — the "pre-activation" of a neuron.
aActivation  a = σ(z) — the output of a neuron after the sigmoid.
σ(z)Sigmoid:  1 / (1 + e⁻ᶻ) — squashes any value into (0, 1).
σ′(z)Sigmoid derivative:  σ(z)·(1−σ(z)) = a·(1−a).
LMSE loss:  ½(y − ŷ)².
δ (delta)Error signal at a layer — gradient of L w.r.t. z at that layer.
ηLearning rate — how large each weight update step is.
// Forward pass for each layer l from 1 to L: z[l] = W[l] · a[l-1] + b[l] a[l] = σ(z[l]) // Backprop (output layer first, then hidden layers) δ[L] = (a[L] − y) · σ′(z[L]) for each layer l from L-1 down to 1: δ[l] = (W[l+1]ᵀ · δ[l+1]) · σ′(z[l]) // Weight update for each layer l: W[l] -= η · δ[l] · a[l-1]ᵀ b[l] -= η · δ[l]
